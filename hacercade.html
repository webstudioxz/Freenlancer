Preguntas Frecuentes
  1. Que es una página web

    Una página web es un documento electrónico accesible a través de internet mediante navegadores que contiene información organizada precisamente sobre el tema seleccionado y presentada de forma visual. Las páginas web se utilizan para diversos propósitos, como: Proporcionar información más detallada. Promover productos, brindar algun servicios Ect.

  2. Que es un sub dominio

    Un sub dominio es una forma de tener una dirección en en intertpara un sitio (web) relacionado, como anexo, a una web principal. Los sub dominios son del tipo: https://sub dominio.dominio.com, éstos apuntan realmente a una carpeta del propio alojamiento que has contratado, pero mostrando su contenido desde el sub dominio.

  3. Que es un Freenlancer

    Un Freenlancer es un profesional autonomo en el área de la informática y sus ramas, Espesificamemte en la creación y desarrollo de app y páginas web usando lenguaje complejos de programación como el Javascript , HTML , CSS , C++ , Java , y muchos más
